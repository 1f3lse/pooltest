7、VxPwn--VxWorks漏洞挖掘相关

8、K8tools--各种工具合集

9、yujianrdpcrack--御剑RDP爆破工具

10、linuxcheck--linux安全检查

11、vulners-scanner--基于主机的漏洞扫描程序的PoC

12、hack_tools_for_me--一些利用工具（严禁做非法用途）

13、emergency-Response-Note--应急响应笔记
